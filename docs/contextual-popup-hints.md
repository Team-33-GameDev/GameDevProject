# Contextual popup hints

The onboarding help is presented as short system-notice windows in the
upper-right corner of the player's viewport. The physical CRT display remains
dedicated to score, quota, time and production statistics.

## Behaviour

- Hints never pause the game or intercept mouse input.
- Every topic is shown once per running game session.
- Normal hints use a queue; quota and preparation notices have priority.
- The window scales from 320 to 520 pixels with the viewport.
- Each notice slides in, remains readable for six seconds and fades out.

## Contexts

The popup explains controls after the boss introduction, then reacts to the
first active quota, preparation phase, shop terminal, factory manager, crowbar
purchase, manual upgrade and Big Button use.
